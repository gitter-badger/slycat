.. _GET Projects:

GET Projects
============
Description
-----------

Returns the list of available projects. The HTML representation provides
the main Slycat user interface.

Requests
--------

Syntax
^^^^^^

::

    GET /projects

Responses
---------

Returns
^^^^^^^

text/html, application/json

--------

^^^^^^^^^^^^^^

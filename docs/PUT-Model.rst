.. _PUT Model:

PUT Model
=========
Description
-----------

Modifies a model. Callers may change the model name, description, state,
result status, progress, and message.

Requests
--------

Syntax
^^^^^^

::

    PUT /models/(mid)

Accepts
^^^^^^^

application/json
